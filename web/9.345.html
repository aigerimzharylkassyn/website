Жолдық әдістер

Сабақтың тақырыбы: Жолдық әдістер

1. Жол әдістері
2. Мәселелерді шешу

Аннотация.Деректердің жол түрі, регистрді түрлендірудің негізгі әдістері.

Әдістері мен функциялары

Біз қазірдің өзінде кейбір кіріктірілген мүмкіндіктермен таныспыз: min(), max(), len(), int(), float() және т.б. әдіс — бұл объектімен тығыз байланысты мамандандырылған функция. Функция сияқты, әдіс жеке тапсырманы орындау үшін шақырылады, бірақ ол белгілі бір объект үшін шақырылады және жұмыс уақытында оның мақсатты объектісі туралы "біледі".

Осылайша: әдіс-бұл объектіге қолданылатын функция. Бұл жағдайда жолға. Әдіс нысанның аты ретінде шақырылады. әдіс_атауы (параметрлер).

Мысалы, s. find ('e') — бір'e' параметрі бар find әдісінің s жолына қолдану.

Деректер жолының әдістерін үш топқа бөлуге болады:

1. Регистрді түрлендіру;
2. Іздеу және ауыстыру;
3. Таңбалардың жіктелуі.

Capitalize () әдісі s жолының көшірмесін қайтарады, онда бірінші таңбада бас әріп бар, ал қалған барлық таңбаларда кіші әріп бар.

Swapcase () әдісі s жолының көшірмесін қайтарады, онда бас әріппен жазылған барлық таңбалар кіші әріпке айналады және керісінше.

Title () әдісі s жолының көшірмесін қайтарады, онда әр сөздің бірінші таңбасы бас әріпке аударылады.

Lower () әдісі s жолының көшірмесін қайтарады, онда барлық таңбалар кіші әріппен жазылады.

Upper () әдісі s жолының көшірмесін қайтарады, онда барлық таңбалар бас әріппен жазылады.

Count әдісі(< sub>, < start>, < end>) бастапқы s жолына < sub> ішкі жолының қиылыспайтын көріністерінің санын есептейді.

Startswith әдісі(< suffix>, < start>, < end>) бастапқы s жолының ішкі жолымен басталатынын анықтайды. Егер бастапқы жол ішкі жолынан басталса, әдіс шын мәнін қайтарады,ал егер жоқ болса, онда жалған.

Endswith әдісі(< suffix>, < start>, < end>) бастапқы s жолының < suffix> ішкі жолымен аяқталатынын анықтайды. Әдіс True мәнін қайтарады егер бастапқы жол және False ішкі жолымен аяқталса.

Find әдісі (< suffix>, < start>, < end>) бастапқы s жолында < sub> ішкі жолының бірінші пайда болу индексін табады. Біз бұл әдісті in операторымен бірге тексере аламыз: берілген жолда кейбір ішкі жол бар ма, жоқ па.

Index әдісі(< sub>, < start>, < end>) find әдісімен бірдей (< sub>, < start>, < end>), егер ол valueerror: substring not found қатесін тудырмаса, егер ішкі жол табылмаса.

Strip () әдісі жолдың көшірмесін қайтарады s онда жолдың басында және соңында тұрған барлық Бос орындар жойылады.

Lstrip () әдісі жолдың көшірмесін қайтарады s жолдың басында тұрған барлық Бос орындар жойылған.

Rstrip () әдісі жолдың көшірмесін қайтарады s онда жолдың соңында тұрған барлық Бос орындар жойылады.

Replace әдісі(< old>, < new>) s көшірмесін < old> ішкі жолының барлық пайда болуымен < new> ауыстырады.
