Python тілі

Python - жоғары дәрежелі кодтың оқылуын
және әзірлеушінің өнімділігін арттыруға
мақсатталған жалпы мақсаттағы
бағдарламалау тілі. Python тілі аз синтаксисті
талап етеді. Бірақ сол уақытта стандартты
кітапханасы үлкен көлемді пайдалы
функцияларды қамтиды.
